# DIO-dashboard-vendas-xbox

## Sobre o Projeto
Este projeto consiste na criação de um dashboard de vendas utilizando Excel, com foco na análise de assinaturas do Xbox Game Pass.

O dashboard foi desenvolvido para apresentar informações de forma simples e visual, permitindo analisar:
- faturamento total: anual, trimestral e mensal;
- vendas de EA Play;
- vendas de Minecraft Season Pass;

---

## Dados Utilizados
A base de dados contém informações sobre:
- assinantes;
- planos;
- tipos de assinatura;
- valores de vendas;
- season pass;
- renovação automática.

---

## Ferramentas Utilizadas
- Microsoft Excel
- Tabelas Dinâmicas
- Gráficos Dinâmicos
- GitHub

---

## Como Reproduzir
1. Baixe o arquivo `.xlsx`
2. Abra no Microsoft Excel
3. Acesse a aba do dashboard
4. Utilize os filtros para visualizar os dados
